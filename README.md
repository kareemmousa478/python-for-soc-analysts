# 🛡️ Python for SOC Analysts

A collection of Python scripts designed to help SOC Analysts automate common cybersecurity tasks.

---

## 📂 Projects

### 1️⃣ IP Address Validator

**Description**
- Validate IPv4 and IPv6 addresses.
- Detect whether an IP address is:
  - Public
  - Private
  - Loopback
  - Multicast

**Skills Used**
- Python
- ipaddress module

---

### 2️⃣ Log Parser

**Description**
- Read log files.
- Extract IP addresses using Regular Expressions (Regex).
- Display unique IP addresses found in logs.

**Skills Used**
- Python
- Regex
- File Handling
